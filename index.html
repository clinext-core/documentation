CliNext framework

An opinionated Node JS CLI framework to help you build modern CLIs. Get ready to change the way you code your CLI! 🚀

Quick StartGuides

Commands ready

CliNext makes it easy to add a command by creating a folder and adding its subcommands.

Options simplified

Options can be defined at a global level and inherited at a command level to promote reuse.

All declarative transformers and validators

Every aspect of a command parameters can be declared easily via input and output transformers, an input validators.

Validators

When features need to communicate the CliNext.Services namespace is ready for inter-feature communication. Any feature can register an api associated with a unique ID that can be called with no knowledge of the feature that handles the operation.

Transformers

CliNext Automatic Seed is an easy way to seed CliNext models. It was built from the ground up to respect the feature lifecycle and separation of concerns. Every feature can define config entries that drive its behaviour at runtime, including the app feature.

Side effects

CliNext config adds the ability to change the behaviour and appearance of your CliNext app without redeploying your server or requiring users to update their app. Every feature can manage its own configuration.
