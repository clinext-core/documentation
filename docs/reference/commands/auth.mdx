# $ clinext auth

## Status
Checks authentication status with CliNext server.
```bash
clinext auth status
```

### Options
N/A

### Example
```bash
clinext auth status
```

### Help
N/A

## Login
Log into CliNext server.
```bash
clinext auth login
```

### Options
```bash
      --registryUsername      Username                                                                            [string]
  -u, --registryPassword      Password                                                                            [string]
      --registrySessionToken                                                              [number] [default: 1387]  
```

### Example
```bash
clinext auth login --registryUsername myemail@gmail.com --registryPassword XXXX-XXXX
```

### Help
```bash
clinext auth login -h
```


## Logout
Logout of CliNext server.
```bash
clinext auth logout
```

### Options
```bash
    --registryConfirmLogout  Logout of CliNext                                                           [number] [default: 1387]  
```

### Example
```bash
clinext auth logout
```

### Help
```bash
clinext auth logout -h
```
