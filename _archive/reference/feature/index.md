# index
Feature's index.

:::info Path
**/index.json**
:::

## Values
- **id**: a unique identifier for the feature
- **version**: the target version for the schema. The desired version should be present in schema

## Example

```json
{
    "id": "localable",    
    "version": "1.0.1"
}
```
