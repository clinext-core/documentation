# Anatomy of a CliNext app

## Project structure
```bash
├── app.js
├── src
├──── app.js
├──── commands
├────── command A
├──────── sub command X
├──────── sub command Y
├────── command B
├──────── sub command Z
├──────── sub command W
├── clinext.config.js
├── .env
└── package.json
```

- The clinext config is clinext.config.js
- The clinext app entry point is index.js
- The app feature is in /app
- The local features are in /features
